# EnaLog Go

### Installation

```bash
go get github.com/enteka-software/enalog-go
```

### Usage

```go

import (
    "os"
    "github.com/enteka-software/enalog-go"   
)

    client, _ := enalog.New(os.Getenv("ENALOG_API_KEY"))
    
    event := enalog.Event{
        Project: "landing-page",
```
